# Git Setup Instructions

To initialize Git and connect to the remote repository, run these commands in your terminal:

## Step 1: Initialize Git Repository
```bash
git init
```

## Step 2: Add Remote Repository
```bash
git remote add origin https://github.com/motycka-creator/cisteniaochrana.git
```

## Step 3: Add All Files
```bash
git add .
```

## Step 4: Create Initial Commit
```bash
git commit -m "Initial commit: Bootstrap 5 website setup"
```

## Step 5: Push to Remote Repository
```bash
git branch -M main
git push -u origin main
```

## Note
If Git is not installed on your system, you can download it from: https://git-scm.com/download/win

After installation, restart your terminal and run the commands above.
